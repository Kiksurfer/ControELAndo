tapunto-voz

Control total del ordenador mediante la voz. Diseñado para personas con movilidad reducida.

Descargar tapunto-voz.exe Windows 10 / 11 · Gratuito · Código abierto

Windows 10/11· Micrófono· Funciona sin internet· ~140 MB

Información sobre uso de inteligencia artificial. Este programa utiliza un sistema de inteligencia artificial (motor Vosk de reconocimiento del habla) para traducir tu voz en comandos. La transcripción la realiza un modelo de aprendizaje automático y, por tanto, puede contener errores. El reconocimiento se realiza íntegramente en tu ordenador: el audio NO se envía a ningún servidor externo y NO se comparte con terceros.
Antes de descargar. Al descargar y usar tapunto-voz aceptas las Condiciones de uso, Política de Privacidad y Aviso Legal. El programa se ofrece sin garantía de funcionamiento. NO es un producto sanitario y NO sustituye atención médica ni terapia ocupacional.

Cómo instalar y verificar

  1. Descarga el archivo
    Haz clic en el botón de descarga. Se descargan dos ficheros: tapunto-voz.exe y tapunto-voz.exe.sha256.
  2. Verifica la firma digital
    Haz clic derecho sobre tapunto-voz.exe → Propiedades → pestaña «Firmas digitales». La firma debe corresponder al titular indicado más abajo en esta página.
  3. Verifica el hash SHA-256
    Abre PowerShell en la carpeta de descarga y ejecuta Get-FileHash -Algorithm SHA256 .\tapunto-voz.exe. El valor debe coincidir con el contenido del archivo .sha256.
  4. Ejecuta y acepta la información
    Doble clic. La primera vez aparece una pantalla con la información previa al uso. Léela con calma y acéptala si estás conforme. Solo entonces se activa el reconocimiento de voz.
Si Windows muestra advertencia de SmartScreen: no la ignores. Comprueba primero la firma y el hash siguiendo los pasos 2 y 3. Si la firma es correcta y el hash coincide, el archivo es auténtico aunque Windows pida confirmación. Si algo no encaja, NO ejecutes el archivo y avisa por correo a contacto@tapunto.app.

Requisitos

  • Sistema operativo: Windows 10 o Windows 11 (64 bits)
  • Micrófono: integrado o externo (recomendado externo con cancelación de ruido)
  • Conexión a internet: NO necesaria para usarlo. Solo se utiliza la primera vez si decides actualizar el modelo de voz.
  • Python: no es necesario instalarlo — el ejecutable lo incluye todo
